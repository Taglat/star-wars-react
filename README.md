## Установка и запуск
1. Копирование репозитория и входим в папку
```
git clone https://github.com/Taglat/star-wars-react.git
cd star-wars-react
```
2. Установка npm пакетов  
```
npm install
```
3. Запуск проекта  
```
npm run dev
```
